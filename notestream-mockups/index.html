About: Mocks for a "Groups" feature in Notestream — letting users share entire streams of notes with teams, families, book clubs. Includes a Slack importer that pulls historical tagged messages into a team group. Designed against the existing Notestream codebase, so the proposed schema and UI integrates with how sharing already works today.

Locked design decisions: name = "Groups", tag binding = explicit via SharedHashtag.defaultGroupId, Slack import scope = tagged messages + reply threads, group stream = explicit shares only.
