Texting-agent mockups

Three self-contained HTML mockups for a planned project: a Claude-backed agent added to iMessage or OpenChat group texts, with consent-mediated disclosure of private data. See the wiki for full planning docs.

  • 00 — Platonic ideal

    UX scenes (attention, the consent-loop split view, manifest dialog, reactions) + abstract architecture diagram + user stories.

  • 01 — Option 2 · Piyush-style

    Concrete 3-machine architecture: Bot Gateway + Workspace Host + noos. SSH + claude --session-id -p per turn.

  • 02 — Option 4 · Noos-style

    Concrete 1-box stateless architecture. Per-turn ephemeral claude --print with transcript rebuilt from SQLite. Cheapest to ship.
